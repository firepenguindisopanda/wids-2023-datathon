## Codebase used when participating in the WiDS 2023 Datathon competition

### The dataset link

[WiDS 2023 Datathon Dataset](https://www.kaggle.com/competitions/widsdatathon2023/data)

### Setup

- Use pyenv or uv or python -m venv venv
- activate the virtual environment
- install packages: `pip install -r requirements`
- create a data folder and store the csv files, which you can download from the link
